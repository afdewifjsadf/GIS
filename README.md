# GIS
Create a `.env` file in the root of your project:

```dosini
PORT="YOURPORTSSERVER"
DBTABLE="YOURDBTABLENAME"
```


Change a `dbConfig` file in the root of your project:

```dosini
user: "YOUUSERNAME",
password: "YOUPASSWORD",
server: "YOURSERVER",
database: 'YOUDBNAME',
options: {
   encrypt: false
},
port: "YOUDBPORT"
```
